### Usage

Simply download and load the CSS in your project.

```html
<link id="stylesheet" rel="stylesheet" href="/thbanklogos.min.css">
```

Then use icons like this

```html
<i class="thbanks thbanks-scb" aria-hidden="true"></i>
```
